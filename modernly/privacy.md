Longyan Privacy Policy
Last updated: 5/12/2026
Longyan is a Chrome extension that translates user-highlighted Chinese text on webpages. This policy explains what data the extension handles.
Data the extension processes

Highlighted text: When you manually highlight Chinese text and request a translation, that text is sent to the Anthropic Claude API for translation. It is not sent to any server operated by the developer of Longyan.
Anthropic API key: You provide your own Anthropic API key, which is stored locally in your browser using Chrome's storage API. It is never transmitted to the developer or any third party other than Anthropic, where it is required to authenticate API requests.
Cached translations: Translations are cached locally in your browser to reduce API costs and latency. This cache is stored on your device only.

Data the extension does not collect
Longyan does not collect, store, or transmit any personally identifiable information, browsing history, location data, or any other user data to the developer. There is no backend server. The developer has no visibility into your usage of the extension.
Third-party services
Highlighted text is sent to the Anthropic Claude API solely for the purpose of translation. Anthropic's handling of API data is governed by their own privacy policy, available at https://www.anthropic.com/legal/privacy.
Data sharing
The developer of Longyan does not sell, share, or transfer any user data to third parties.
